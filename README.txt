KRCS SURGE SYSTEM - GENERATED FILES

MANUAL STEPS REQUIRED:
1. Leaflet JS/CSS: 
   - Download 'leaflet.css' and put it in: krcs_surge/public/css/
   - Download 'leaflet.js' and put it in: krcs_surge/public/js/
   
2. Map Data:
   - Place your 'kenya_counties.json' GeoJSON file in: krcs_surge/public/js/

3. Deployment:
   - Upload this 'krcs_surge' folder to your bench's 'apps' directory.
   - Run 'bench install-app krcs_surge'
